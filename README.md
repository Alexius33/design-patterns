## Showcases for design patterns implemented in Java
### Includes:

- Behavioural patterns:
  - Chain of Responsibility pattern
  - Command pattern
  - Mediator pattern
  - Observer pattern
  - State pattern
  - Template method pattern
- Creational patterns:
  - Factory method pattern
  - Singleton pattern
- Structural patterns:
  - Adapter pattern
  - Composite pattern
  - Decorator pattern
  - Facade pattern
  - Proxy pattern
